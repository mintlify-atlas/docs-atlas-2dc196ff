TDengine TSDB is a time-series database designed to help traditional industries overcome the challenges of Industry 4.0 and Industrial IoT. It securely ingests, stores, analyzes, and distributes petabytes of data per day generated by billions of sensors and devices. With a built-in AI agent for time-series forecasting and anomaly detection, TDengine TSDB delivers real-time business insights.

## Core capabilities

TDengine's "one table per device" design, unique supertable concept, and highly optimized storage engine make it purpose-built for massive time-series workloads.

### Data ingestion and querying

Write data using standard SQL or in schemaless mode over the InfluxDB Line Protocol, OpenTSDB Telnet Protocol, or OpenTSDB JSON Protocol. TDengine also integrates seamlessly with Telegraf and Prometheus.

For querying, TDengine extends standard SQL with time-series functions like downsampling, windowing, cumulative sum, and time-weighted average. User-defined functions (UDFs) can be written in C or Python.

### Built-in stream processing

TDengine's built-in stream processing engine processes data streams in real time as they are written. It supports continuous queries and event-driven stream processing, returning results in milliseconds even during high-throughput ingestion — without any separate streaming infrastructure.

### Read caching

TDengine uses a time-driven FIFO cache strategy, keeping the most recent data in memory. This provides instant access to the real-time status of any metric without deploying Redis or another caching layer, simplifying your architecture and reducing operational costs.

### Data subscription

Define topics in SQL — subscribing to a query, supertable, or database — and consume them with a Kafka-like API. No separate message broker required.

### TDgpt AI agent

TDengine includes a built-in AI agent for time-series forecasting and anomaly detection, delivering real-time business insights directly from your data pipeline.

### Ecosystem integrations

Through its REST API and standard JDBC and ODBC interfaces, TDengine integrates with Grafana, Power BI, Seeq, and other leading visualization and BI platforms.

## Getting started

<Steps>
  <Step title="Choose your deployment">
    Run TDengine in [Docker](/get-started/docker) for the fastest local setup, install it from a [package](/get-started/package) on Linux or Windows, or sign up for [TDengine Cloud](/get-started/cloud) for a fully managed experience with no installation required.
  </Step>
  <Step title="Connect with the taos CLI">
    Once TDengine is running, open the interactive command-line interface by running `taos` in your terminal. Use it to create databases, run queries, and manage your cluster.
  </Step>
  <Step title="Ingest your data">
    Write data using standard SQL, the InfluxDB Line Protocol, or zero-code connectors for MQTT, OPC, PI System, and other industrial sources.
  </Step>
  <Step title="Query and visualize">
    Run time-series queries in SQL and connect Grafana or Power BI to build dashboards and alerting on top of your data.
  </Step>
</Steps>
